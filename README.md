# Swapnil Mogal – Portfolio

A modern, single-page portfolio website that showcases my journey as a Data Science and Machine Learning Engineer. The site is built with Next.js 14 and focuses on clean design, smooth animations, performance, and a warm professional aesthetic.

## Overview

This portfolio serves as a central place to present my professional background, technical skills, projects, internships, and certifications. It is designed as a single-page application with smooth scrolling and responsive layouts to ensure a seamless user experience across devices.

## Key Features

* Fully responsive design compatible with desktop, tablet, and mobile devices
* Modern and minimal UI with a warm, professional color palette
* Single-page layout with smooth scrolling navigation
* Well-structured portfolio sections, including:

  * Professional summary and personal interests
  * Technical skills grouped by category
  * Internship experience at NPCI, Artizence, and Finlatics
  * Seven GitHub projects with live repository links
  * Blog posts embedded from Medium
  * Embedded resume viewer with download option
  * Professional certifications
  * Leadership experience and community involvement
* Performance-optimized using Next.js App Router
* Smooth fade-in animations triggered on scroll
* Direct resume download in PDF format

## Tech Stack

* Framework: Next.js 14 (App Router)
* Language: TypeScript
* Styling: Tailwind CSS v4
* Deployment: Vercel
* Version Control: Git and GitHub

## Project Structure

```
app/
 ├── layout.tsx        # Root layout
 ├── page.tsx          # Main portfolio page
 ├── components/       # Reusable UI components
 ├── sections/         # Portfolio sections (About, Skills, Projects, etc.)
 ├── styles/           # Global styles
 └── assets/           # Images, icons, and static files
```

## Performance and Design Philosophy

The portfolio is designed with a strong emphasis on performance, accessibility, and clarity. The Next.js App Router ensures fast page loads, while Tailwind CSS enables consistent styling with minimal overhead. Animations are kept subtle to enhance user experience without affecting performance.

## Deployment

The application is deployed on Vercel for fast global delivery and seamless CI/CD integration. Every push to the main branch triggers an automatic production deployment.

## Purpose

This portfolio is intended to represent my technical capabilities, problem-solving mindset, and professional growth as a Data Science and ML Engineer. It is suitable for recruiters, hiring managers, and collaborators looking to understand my work and experience at a glance.
